# CBC Learner Performance Analysis

This project presents a CBC-compliant learner performance analysis using
Python and Jupyter Notebook.

## Features
- Learner-level analysis
- CBC performance tallies
- Data visualizations
- Professional reporting format

## Tools Used
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Purpose
Academic reporting, CBC analytics, and educational data analysis.
